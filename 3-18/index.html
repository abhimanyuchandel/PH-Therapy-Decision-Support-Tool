Hemodynamics and Perfusion

Category D1 reflects transient hypotension. Category D2 reflects normotensive shock with isolated hypoperfusion markers such as lactate >2 mmol/L, AKI, urine output <0.5 mL/kg/h, mental status change, cardiac index <2.2 L/min/m2, MAP <60 mm Hg, or increased shock score/stage.

RV / Biomarkers / Respiratory Status

Respiratory modifier: for category C, add R with O2 saturation <90%, RR >=30, or any supplemental oxygen requirement. For categories D-E, add R only with >6 L NC/nonrebreather, HFNC, noninvasive ventilation, or mechanical ventilation.
